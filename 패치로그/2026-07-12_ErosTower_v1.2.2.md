## 변경 내용

1. 앨범 사진 상세 화면의 내부 스크롤을 제거했다.
2. 제목, 사진, 글귀를 서로 떨어진 고정 행이 아니라 하나의 묶음으로 화면 중앙에 배치했다.
3. 사진은 원본 비율을 유지하며 앨범 창의 남은 높이와 너비 안에서 자동 축소된다.
4. 제목, 사진, 글귀 간격은 데스크톱 12px, 모바일 8px로 유지된다.
5. 가로형과 세로형 사진 모두 모바일 화면 안에서 중앙 정렬되도록 했다.

## 정식 승격

- 제품 저장 접두사 `eros_tower_v02:`를 유지한다.
- UI ID `eros-tower-v03-*`를 유지한다.
- 테스트 전용 저장공간, UI ID, 표시명, 업데이트 URL은 포함하지 않는다.
- 앨범 외 Provider, 에이전트, Psyche, canonical, 상태, 기억, 번역, 이미지 생성 구조는 변경하지 않는다.

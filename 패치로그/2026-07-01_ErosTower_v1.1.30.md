# Eros Tower v1.1.30

- 번역 에이전트 설정의 `빈 응답 재시도` 입력칸을 줄이고, 옆에 `원문 병행` 체크박스를 추가했다.
- `원문 병행`은 기존 번역 프롬프트 프리셋을 수정하지 않고, 체크된 호출에만 병행 출력용 형식 지시를 덧붙인다.
- `원문 병행` 사용 시 번역 대상 본문을 문단/대사 단위로 나누어 `id`별 JSON 배열로 한 번만 전송한다.
- 번역 에이전트 응답의 `target_text`를 플러그인이 로컬에서 `[원문]` / `[번역]` 교차 형식으로 조립한다.
- 빈 응답 재시도 기능은 유지했고, 타임아웃 설정은 변경하지 않았다.

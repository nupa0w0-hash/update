# Eros Tower v1.1.37

## Psyche Weave 기반 추가
- `psycheWeave` 상태 계층을 추가했습니다.
- 원천 로어, 내부 `psycheUnits`, 관리상태, 장기기억, 비밀, 관계, 세계 전선을 하나의 내부 그래프 레이어로 엮습니다.
- 이 레이어는 옴니노드/HME Graph식 노드-엣지 구조를 참고하되, 에로스 타워의 관리상태/관점/비밀/에이전트 노트와 함께 작동하도록 별도 구현했습니다.

## 검색/주입 개선
- 기존 `associationGraph` 점수에 더해 `Psyche Weave` 활성도를 검색 점수에 반영합니다.
- 메인 모델 주입에 `[Psyche Weave]` 브리핑 블록을 추가했습니다.
- 에로스 에이전트 컨텍스트팩에도 같은 Weave 브리핑을 제공해, 에이전트와 메인 모델이 같은 내부 세계망을 참고하도록 했습니다.
- 비밀/관점 경계는 기존 knowledge boundary를 그대로 적용하여, `cannotKnow` 관점의 노드가 무분별하게 전파되지 않도록 했습니다.

## 실행 흐름
- 전처리 전 1회, 전처리 에이전트 노트 생성 후 1회, 후처리 관리상태 커밋 뒤 1회 Weave를 갱신합니다.
- 새 API 호출을 추가하지 않고, 이미 수집/추출된 상태를 로컬에서 엮습니다.
- Run Log에는 본문을 늘리지 않고 Weave 노드/엣지 숫자만 기록합니다.

## 진단/대시보드
- 상태 대시보드에 `Psyche Weave` 노드/엣지 카운트를 추가했습니다.
- 자동 기억 엔진 패널에서 Weave 전파 단계와 통계 요약을 볼 수 있게 했습니다.
- 스냅샷 저장 시 Weave 전체를 과하게 저장하지 않고 일부 노드/엣지만 보존하도록 제한했습니다.

## 기타
